WASD (or arrow keys) to move
X to attack
R to reset the level
Hold 1 for god mode (invulrable + flight)
The goal is to find the goblet
Every time the stage is beaten, the number of rooms increses by 5
Every time the player dies the number of rooms is reset to 5
